# COFFEE-DISEASE-DETECTION
Coffee leaf disease detection system using YOLO for real-time identification and localization of infected coffee leaves.

Datasets Used:
1. https://www.kaggle.com/datasets/mohammedzwaughfa/coffee-leaf-disease-dataset
2. https://www.kaggle.com/datasets/tuannguyenhoang/coffee-leaves-disease-5-classes-merged-dataset
